# copyaddrs

[![Build Status][build-badge]][build-url]

copyaddr is Thunderbird WebExtension.

## Develop

There is an npm package available, which means you can grab it from there:

```sh
# yarn version
yarn install
yarn test
yarn build

# npm version
npm install
npm run test
npm run build
```
